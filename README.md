# 📊 Customer Churn Analysis Dashboard – Power BI

## 📌 Project Overview
This project analyzes customer data from a telecom company to understand customer churn behavior and identify factors that influence customer retention. An interactive Power BI dashboard was developed to monitor churn trends and provide insights for improving customer retention strategies.

---

## 📊 Dataset
**Source:** Telco Customer Churn Dataset (Kaggle)

The dataset includes:
- Customer demographics
- Account information
- Services subscribed
- Billing details
- Churn status

---

## 🎯 Objective
- Measure customer churn rate
- Identify key factors influencing churn
- Segment high-risk customers
- Provide insights to improve retention strategies

---

## 🔧 Tools & Technologies Used
- Power BI
- Microsoft Excel (Initial data review)
- GitHub

---

## 📈 Key Metrics Tracked
- Total Customers
- Churned Customers
- Churn Rate (%)
- Average Monthly Charges
- Average Customer Tenure

---

## 📊 Dashboard Features
- KPI cards highlighting overall churn performance
- Churn analysis by contract type
- Tenure-based churn trends
- Monthly charges impact on churn
- Customer segmentation by gender, senior citizen status, and payment method
- Interactive filters for deeper analysis

---

## 🔍 Key Insights
- Customers with **month-to-month contracts** have the highest churn rate.
- Customers with **short tenure** are more likely to leave.
- Higher **monthly charges** are associated with increased churn.
- Customers using **automatic payment methods** tend to churn less.
